# GuidrGo AI

## Context-Aware Travel Recommendation System

GuidrGo AI is an AI-powered travel recommendation platform that combines:

- NLP Sentiment Analysis
- Weather Risk Analysis
- Traffic Risk Analysis
- Hotel Recommendation Engine
- Travel Itinerary Generation
- Context-Aware Risk Assessment

## Features

### Hotel Recommendation
Hotels are ranked using:
- Google Ratings
- User Reviews
- NLP Sentiment Scores

### Risk Analysis
- Weather Conditions
- Traffic Conditions
- Overall Travel Risk

### Travel Planning
- Personalized Itineraries
- Hotel Recommendations
- Travel Reports

## Technologies Used

- Python
- Pandas
- TextBlob
- Google Maps API
- WeatherAPI
- Streamlit
